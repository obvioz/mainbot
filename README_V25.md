# v25 Market Memory + Performance

Что добавлено:

- кэш рынка: повторный `/scan` не дергает все API заново каждую секунду;
- защита от параллельных `/scan`: если один скан уже идет, второй не запускается;
- постоянная память рынка в `data/market_snapshots.jsonl`;
- `/changes` — сравнение текущего/предыдущего скана: цена, score, статус;
- `/memory` — состояние сохраненной памяти рынка;
- `/scan` автоматически сохраняет снимок рынка;
- после `/scan` бот показывает короткий блок изменений с прошлого скана.

Рекомендуемые настройки `.env`:

```env
MARKET_CACHE_TTL_SECONDS=60
MARKET_CONTEXT_CACHE_TTL_SECONDS=180
SAVE_MARKET_SNAPSHOTS=true
```

Важно:

- Это не меняет торговую логику Decision Engine.
- Это не удаляет старые кнопки `Я купил` и ручной `/buy` — мы договорились оставить UI без больших изменений на период тестирования.
- После тестов можно будет сделать отдельную чистку интерфейса.
